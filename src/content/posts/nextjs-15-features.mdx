Next.js 15 is here, and it brings a lot of exciting features to the table. From improved caching defaults to the new `@next/env` package, there's a lot to explore.

## Key Features

1. **Improved Caching**: Defaults are now more intuitive.
2. **React 19 Support**: Full compatibility with the latest React features.
3. **Turbopack**: Even faster dev builds.
